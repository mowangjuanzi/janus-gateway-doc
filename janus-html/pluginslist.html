Out of the box, Janus comes with a set of different and heterogeneous media manipulation plugins. These can be used individually or composed together at an application level for building complex WebRTC-based media applications.

The list of plugins currently available is the following. Since new plugins may become available available in the future, make sure you come back to this page for more information.
